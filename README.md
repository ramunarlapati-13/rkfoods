# Ramakrishna Foods (RK Foods) E-commerce Platform

**Ramakrishna Foods (రామకృష్ణ ఫుడ్స్)** is an elite, heritage-first culinary brand dedicated to delivering traditional Telugu pickles, condiments, and premium sweets. This repository contains the source code for the digital flagship platform bridging generations of pure taste and household trust with high-performance e-commerce architecture.

## 🚀 Live Demo & Domains
- **Target Domain:** [ramakrishnafoods.telugu.in](http://ramakrishnafoods.telugu.in)
- **Staging / Live Events:** [vnlivevents-rexploretech.vercel.app](https://vnlivevents-rexploretech.vercel.app/)

- **Client Support Lead:** RSMK (R. Srinivasa Manikanta)

## 📦 Features
- **Traditional Telugu Pickles & Sweets:** High-quality, authentic products from various Telugu regions.
- **Product Categorization:** Supports robust filtering including Veg and Non-Veg modes for meals and pickles.
- **"Spice & Sweetness" Heat Slider:** Interactive flavor intensity indicator for every product.
- **"Build Your Heritage Box" Sampler Module:** A drag-and-drop interface to customize multi-packs.
- **Direct-to-Cloud User Reviews:** Support for unboxing photos, tasting videos, and audio feedback via Firebase.
- **Comprehensive Admin Dashboard (CRM):** Product management, bulk catalog imports (.xlsx, .docx), and location management.
- **User Checkout Flow:** Integrated Google Auth and Mobile OTP verification with a mandatory user details form.

## 🛠️ Tech Stack & Architecture
- **Frontend Framework:** Next.js (SSR) & Tailwind CSS
- **Media Cloud Storage:** Firebase Storage
- **Primary Database:** Cloud Firestore / MongoDB
- **Deployment Pipeline:** Vercel for continuous integration and hosting
- **Structure:** Divided into `/frontend` and `/backend`

## 🎨 Visual Identity
The platform implements a premium, high-contrast aesthetic rooted in authentic Telugu culture:
- **Primary Deep Clay/Terracotta Red (#8B261E):** Representing traditional earthen pots.
- **Secondary Warm Ochre/Mustard Yellow (#E5A93C):** Reflecting cold-pressed oils and turmeric.
- **Background Matte Charcoal/Dark Slate (#121212):** Providing high contrast for food imagery.
- **Neutral Warm Cream/Off-White (#FAF9F6):** For structural panels and readability.
